# Hypertension Data Analysis

This repository contains data preprocessing and visualization work for the
High Blood Pressure Chatbot project.

## Dataset Features
- Age
- Salt_Intake
- Stress_Score
- BP_History
- Sleep_Duration
- BMI
- Medication
- Family_History
- Exercise_Level
- Smoking_Status
- Has_Hypertension

## Tasks Completed
- Data cleaning
- Handling missing values
- Removing duplicates
- Encoding categorical variables
- Data visualization
- Correlation analysis

## Output
- Cleaned dataset
- Visualizations
- Insights for chatbot recommendations
